Uzart Reporting Module

This directory contains the legacy PHP implementation used for generating and displaying server reports. The PHP entry point lives in report/uzart/index.php, while various configuration and helper scripts reside in the surrounding folders.

Usage

  1. Ensure PHP is installed on your system.
  2. From the project root, serve the report/uzart directory via your preferred PHP-enabled web server.
  3. Access /uzart/ in your browser to view the reporting dashboard.

The Java implementation of Uzart can be found in the java directory. See the project README for build instructions.
